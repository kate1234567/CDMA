# Моделирование CDMA

## Описание

Проект представляет собой консольное приложение на C#, моделирующее передачу данных по технологии CDMA (Code Division Multiple Access).

В программе реализованы генерация кодов Уолша, кодирование сообщений, объединение сигналов в общем канале и последующее восстановление исходных сообщений.

Проект был выполнен в рамках лабораторной работы по дисциплине, связанной с телекоммуникационными технологиями.

## Используемые технологии

- C#
- .NET
- LINQ

## Возможности

- генерация кодов Уолша;
- перевод текстовых сообщений в бинарное представление;
- кодирование сообщений с использованием кодов Уолша;
- объединение нескольких сигналов в одном канале;
- декодирование сообщений;
- вывод промежуточных и итоговых результатов в консоль.

## Запуск проекта

1. Открыть решение `CDMA.sln` в Visual Studio.
2. Запустить проект.
3. Результаты кодирования, объединения сигналов и декодирования будут выведены в консоль.
